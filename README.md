# Resumen del proyecto

**Blog Minimalista con Javascript**: Este proyecto trata sobre como se puede desarrollar un pequeño Blog usando HTML, CSS, JavaScript y archivos Json como repositorio de datos.

# Objetivos del Proyecto

Este proyecto, sigue **dos objetivos** de aprendizaje:

1. Diseñar y codificar un blog utilizando HTML, CSS y Javascript. 
2. Entender los conceptos básicos manipulacion del DOM.

Al final de este proyecto, se podrá comprender cómo insertar componentes HTML dentro del DOM en tiempo real, para hacer una aplicacion SPA dinámica, sencilla y sin la implementación de Frameworks de Front End.

# Requisitos

El único requisito es tener conocimientos básicos en HTML, CSS y Javascript.

# Requerimientos

- Servidor o Servicio Web.

# Estructura del proyecto

El proyecto práctico de **Blog Minimalista con Javascript** se divide en las siguientes tareas:

- Crear los archivos HTML, CSS, Js y Jason con su editor de preferencia.
- Definir el diseño visual de la página.
  - Temas visuales (Dark y Light) con CSS.
  - Posición de elementos en la página con javascript.
  - Implementacion de Media Queries para conseguir diseño Responsive.
- Codificar las funciones para botones y enlaces en Javascript.  
- Ejecutar el codigo, probar y corregir errores.
